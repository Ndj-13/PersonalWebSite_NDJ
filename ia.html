Machine Learning

Program that implements different searching algorithms to find the goal by searched-based agents and machine learning.

Technical Sheet

This project is divided in two excercises.
1. A-Star algorithm

It implements the A Star Mind algorithm that searches Offline a path to the goal. This algorithm just requires the following methods:

  • Search(): starting from the character position node, it searches between it's neightbours the best heuristic until getting to the goal
  • GetNextMove(): starting from the goal, it goes back to the start, following the best heuristics calculated and saving them in a stack that once is completed, return the next move to the character.
  • Repath(): it cleans the stack.
2. Q-learning

It implements the Q-learning algorithm, a reinforcement learning method that motivates the agent to take a decision above an action according to the actual state. It requires the following methods:

  • Learning(): a random cell and action are selected and the agent calculates the Q value according to the learning rule. This process is repeatted until the machine has gone through the table several times to get the best Q values, that are saved in a table.
  • Explote(): moves the character util the goal, following on each cell the best Q value action.
